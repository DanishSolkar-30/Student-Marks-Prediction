# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project predicts the price of a house based on its area using the Linear Regression algorithm from Scikit-learn. A sample dataset is used to train the model, which then predicts the price for a user-provided house area.

---

## 🚀 Features
- Created a dataset containing house areas and their corresponding prices.
- Trained a Linear Regression model using Scikit-learn.
- Accepted area as input from the user.
- Predicted the estimated house price.
- Displayed the prediction in an easy-to-understand format.

---

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

---

## 🤖 Algorithm Used
**Linear Regression**

Linear Regression is a supervised machine learning algorithm used to predict continuous values by finding the best-fit relationship between the input feature (Area) and the target value (Price).
